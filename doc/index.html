Instruções para uso do modelo de dissertação/tese do IMECC

Bem-vindo a página inicial sobre o modelo em LaTeX de dissertações e teses do IMECC. Informações sobre as normas da UNICAMP sobre dissertações e teses encontram-se aqui e uma lista de editores para LaTeX aqui.

Informações sobre a organização do modelo encontram-se disponíveis aqui.

Para gerar o PDF a partir do código LaTeX fornecido nesse modelo você irá precisar de um a distribuição do LaTeX. Para saber como instalar as distribuições existentes, veja a lista abaixo:

  • TeX Live (Linux) (Windows 7) (Windows 8) (Mac OS X),
  • MiKTeX (Windows 7) (Windows 8).

Existe uma lista de perguntas frequentes aqui e perguntas não existentes nessa lista (bem como dúvidas, problemas, sugestões e elogios) podem ser enviados para modelo-latex-l@listas.unicamp.br (uma lista de email hospedada no CCUEC na qual todos são convidados a se inscrever).
