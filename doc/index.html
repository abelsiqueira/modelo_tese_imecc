Instruções para uso do modelo de dissertação/teses do IMECC

Os arquivos que compõe o modelo de dissertação/teses do IMECC encontra-se disponíveis em https://github.com/r-gaia-cs/modelo_tese_imecc.git (download).

Para obter o PDF referente a dissertação/tese do IMECC você irá precisar de um a distribuição do LaTeX. Para saber como instalar as distribuições existentes, veja a lista abaixo:

  • TeX Live (Linux) (Windows 7) (Windows 8) (Mac OS X),
  • MiKTeX (Windows 7) (Windows 8).

O modelo de dissertação/tese do IMECC foi feito utilizando a codificação UTF-8 e para não ter problemas você terá que configurar seu editor de maneira adequada. Para saber como configurar seu editor, veja a lista abaixo:

  • TeXworks (não requer nenhuma configuração),
  • Texmaker,
  • TeXnicCenter 2 (não requer nenhuma configuração e a versão 1 não é suportada).

Alguns leitores de PDF impedem que o mesmo seja alterado por outro programa enquanto o mesmo estiver sendo visualizado. Para saber como configurar sue leitor de PDF e evitar isso, veja a lista abaixo:

  • TeXworks built-in PDF viwer (não requer nenhuma configuração),
  • Evince (não requer nenhuma configuração),
  • SumatraPDF (não requer nenhuma configuração),
  • Foxit PDF Reader,
  • Adobe Reader.

Dúvidas, problemas, sugestões e elogios devem ser enviados para teseimecc@ime.unicamp.br.
